# Companion Code Reference

These are the key implementations for each lesson. When teaching, use these as the source of truth for what the code should look like. Do NOT improvise implementations.

Each file contains the complete, tested code for that lesson's primary artifact. Read the relevant file before teaching the lesson.

| File | Lesson | What it contains |
|------|--------|-----------------|
| `00-claw.mjs` | 00 Substrate | The claw script with execFileSync, sessions, cwd |
| `01-skill.md` | 01 First Skill | The gws SKILL.md with real CLI syntax |
| `02-gather.mjs` | 02 Awareness | Gatherer with adapters, Haiku, digest caching |
| `02-email-adapter.mjs` | 02 Awareness | Email adapter with real gws API calls |
| `02-calendar-adapter.mjs` | 02 Awareness | Calendar adapter with real gws API calls |
| `04-memory.mjs` | 04 Memory | Memory analyzer with Sonnet, 3-interaction threshold |
