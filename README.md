# wizard-skills

Companion skills for [wizardshit.ai](https://wizardshit.ai) courses.

## Install a skill

```bash
npx skills add badass-courses/wizard-skills --skill <skill-name>
```

## Available skills

| Skill | Course |
|-------|--------|
| `my-claw-with-claude-code-companion` | [My Claw: Build Your AI System with Claude Code](https://wizardshit.ai/my-claw-with-claude-code) |
